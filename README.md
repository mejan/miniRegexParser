miniRegexParser
===============
course in programming language theory.

And as the title says it's a small version of a regex parser.

BNF
===
Expr => Concat | Paren | Repeat | Or

Or => Concat + Concat

Parentes => (Expr)

Concat => Id | Id-Concat

Repeat => Id* | Parentes*

Id is a charactor/number
